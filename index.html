Funktionen

Alles was Ihre Bücherei braucht

OpenLibry bietet alle wichtigen Funktionen für eine effiziente Büchereiverwaltung – übersichtlich, schnell und ohne unnötigen Ballast.

Schnelle Ausleihe

Bücher ausleihen und zurückgeben mit nur einem Klick. Barcode-Scanner werden unterstützt.

Bücherverwaltung

Bücher hinzufügen, bearbeiten und durchsuchen. Automatische Metadaten über ISBN.

Nutzerverwaltung

SchülerInnen und Klassen verwalten. Import aus Excel und Klassenversetzung inklusive.

Reports & Statistiken

Übersicht über ausgeliehene Bücher, Mahnungen und Statistiken. Export als CSV.

DSGVO-konform

Alle Daten bleiben bei Ihnen – keine Cloud, keine externen Server. Lokal installiert.

Open Source

Quellcode unter MIT-Lizenz auf GitHub. Anpassbar und von der Community unterstützt.

Installation mit Docker

OpenLibry lässt sich mit wenigen Befehlen auf jedem System mit Docker installieren.

# Repository klonen $ git clone https://github.com/jzakotnik/openlibry.git $ cd openlibry # Container starten $ docker compose up -d # Fertig! Öffnen Sie http://localhost:3000

Schnellstart in 3 Schritten

  1. 1Docker Desktop installieren
  2. 2Repository klonen und docker compose up ausführen
  3. 3Im Browser localhost:3000 aufrufen – fertig!
