Installation mit Docker

OpenLibry lässt sich mit wenigen Befehlen auf jedem System mit Docker installieren. Hier sehen Sie, wie schnell Sie starten können:

# Repository klonen $ git clone https://github.com/jzakotnik/openlibry.git $ cd openlibry # Container starten $ docker compose up -d # Fertig! Öffnen Sie http://localhost:3000

Schnellstart in 3 Schritten

  1. 1 Docker Desktop installieren (falls noch nicht vorhanden)
  2. 2 Repository klonen und docker compose up ausführen
  3. 3 Im Browser localhost:3000 aufrufen – fertig!
